📦 FirstPen Waitlist Widget — 埋め込みサンプル

既存のページ (例: firstpen-platform.vercel.app/waitlist.html) に以下のコードを貼るだけで動作します。

① 最小構成 (ダークテーマ)

<div data-firstpen-form></div>
<script src="https://firstpen-waitlist-api.YOUR-SUBDOMAIN.workers.dev/widget.js" defer></script>

② ライトテーマ + カスタムタイトル

<div data-firstpen-form
     data-theme="light"
     data-title="ローンチ通知を受け取る"
     data-subtitle="先着300名限定の特典付き"></div>

③ 同一ページに複数配置も可能

<!-- ヒーロー直下 -->
<div data-firstpen-form></div>

<!-- ページ下部に再表示 -->
<div data-firstpen-form></div>

属性一覧

属性値説明
data-firstpen-form(必須)このdivにフォームが描画されます
data-themedark / lightカラーテーマ (デフォルト: dark)
data-title任意文字列フォーム上部のタイトル
data-subtitle任意文字列サブタイトル

動的にフォームを追加するには

// JavaScript で動的にDOMを追加した後
const div = document.createElement('div');
div.setAttribute('data-firstpen-form', '');
document.body.appendChild(div);
window.FirstPen.mount(div);  // 単一要素にマウント
// または
window.FirstPen.init();      // ページ内の全 [data-firstpen-form] を再スキャン
